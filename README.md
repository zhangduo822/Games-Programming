# Unity Time Replay Demo

This folder contains an implementation-ready MVP for a 2D time replay puzzle demo.

## 1) Create the Unity project from Unity Hub

1. Open Unity Hub.
2. Create a new project with `Unity 2022.3 LTS` and template `2D Core`.
3. Name it `unity-time-replay-demo`.
4. Copy this folder's `Assets` contents into your Unity project's `Assets` folder.

## 2) Controls

- `A / D` or arrow keys: move
- `Space`: jump
- `R`: start recording or stop recording and spawn replay clone
- `T`: quick reset
